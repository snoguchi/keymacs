=========================================================================
 Keymacsのソース
=========================================================================
■このソースについて
  　これはKeymacs(Windows用キーボードユーティリティー)のソースコードです．
  (少しだけ)要望があったので公開しますが，なにぶんC言語始めて日が浅いので，
  めちゃくちゃな部分もあるかと思います．バグや，「こんな書きかたすんじゃねー」
  等のお叱りがあれば，メールにてご指摘ください．

  　ちなみに私の開発環境は，VC++ 6.0 on Win98です．他の環境では，そのまま
  ではうまくいかないと思います．


■ファイル構成(重要なもの)
  ・Keymacs.c		Keymacs.exeのメイン部分
  ・KeymacsProp.c	Keymacs.exe設定ダイアログ関係
  ・dll/keymacsDll.c	Keymacs.dll関係


■注意
  　配布しているバイナリファイルは，tiny.objを利用してサイズを小さしてあり
  ます．このソースをそのままコンパイルしても，サイズが異なるはずです．機能
  的には変わりありません．tiny.objのライセンスが良くわからなかったので，と
  りあえず除いておきました．
